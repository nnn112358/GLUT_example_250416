

## プログラムの説明

このプログラムは、OpenGLとGLUTを使用して三角形を描画し、マウスで回転操作ができるものです。

### 主な機能

1. **三角形の表示**
   - 中央に赤色の三角形を描画します

2. **マウス操作**
   - 左ボタンドラッグ: 三角形を回転させる
   - 右ボタンドラッグ: ズームイン・ズームアウト

3. **キーボード操作**
   - `q`: プログラム終了
   - `c`: 視点を中心に移動
   - `r`: 視点をリセット
   - 矢印キー: 視点の移動
   - Page Up/Down: Z方向の移動

### コンパイル方法

1. ターミナルで以下のコマンドを実行:
   ```
   make
   ```

2. 実行ファイルが生成されたら、以下のコマンドで実行:
   ```
   ./triangle
   ```

### 必要なライブラリ

- GL/GLUT
- GL/GLU

Ubuntuなどのシステムでは、以下のコマンドでライブラリをインストールできます:
```
sudo apt-get install freeglut3-dev
```
